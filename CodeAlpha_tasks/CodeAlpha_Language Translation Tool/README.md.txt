# 🌍 Text Translation Tool

A simple and user-friendly **Text Translation Tool** built using **Python** and the **Google Translate API**. This project allows users to translate text from one language to another with ease through a command-line interface.

## 📌 CodeAlpha Internship Project

**Project Title:** Text Translation Tool
**Internship:** CodeAlpha Artificial Intelligence Internship

---

## 📖 Project Overview

The Text Translation Tool is designed to break language barriers by translating text into multiple languages. It uses the Google Translate service to provide fast and accurate translations.

This project demonstrates the practical use of APIs in Python while building a simple AI-powered application.

---

## ✨ Features

* 🌐 Translate text into 100+ languages
* ⚡ Fast and accurate translations
* 🖥️ Easy-to-use command-line interface
* 🔄 Supports automatic language detection (optional)

## 🚀 Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/text-translation-tool.git
```

### 2. Navigate to the project folder

```bash
cd text-translation-tool
```

### 3. Install the required library

```bash
pip install -r requirements.txt
```

---

## ▶️ Usage

Run the Python file:

```bash
python translator.py
```

Enter:

* Text to translate
* Source language code (or `auto`)
* Target language code

Example:

```
Enter text: Hello, how are you?
Source language: en
Target language: hi

Translated Text:
नमस्ते, आप कैसे हैं?
```
